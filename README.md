# Baroque Fashion Website UI

This project is a responsive fashion website inspired by the Baroque clothing brand.

The website is built using HTML and CSS and focuses on modern layout design and responsive web design principles.

## Live Website

https://astonishing-yeot-e771de.netlify.app/

## Features

- Responsive Design (Mobile, Tablet, Desktop)
- Hero Banner Section
- Fashion Collection Sections
- Grid Image Layout
- Modern Navigation Bar
- Footer with Social Icons
- Clean UI Design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Remix Icons

## Sections Included

- Top Announcement Bar
- Navigation Bar
- Hero Banner
- Own Your New Look Section
- Eid Lawn Collection
- Chantelle Collection
- Solid Essentials
- Footer Section
